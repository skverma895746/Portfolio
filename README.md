# Personal Portfolio Website

A responsive personal portfolio website built with HTML, CSS, and JavaScript. It showcases profile details, education, skills, projects, social links, a contact form, and a theme toggle.

## Preview

This portfolio includes:

- Sticky navigation with section tracking
- Responsive layout for desktop and mobile
- Theme toggle with a premium alternate color palette
- Skills section with animated progress rings
- Dynamic project cards rendered with JavaScript
- Contact form integration using Formspree

## Main Files

- `index.html`: Main page structure for the portfolio
- `assets/css/index.css`: Styling, responsive layout, and theme colors
- `assets/js/index.js`: Theme toggle, navbar behavior, skill animation, project rendering, and contact form handling
- `assets/images/`: All portfolio images and skill/project assets

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Formspree

## How To Run

1. Download or clone this repository.
2. Open the project folder.
3. Run `index.html` in your browser.

You can also use VS Code Live Server for a smoother development experience.
